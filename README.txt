Overview
Pelanthia is a responsive and elegant website designed to showcase handcrafted floral arrangements. It includes a curated image carousel, smooth navigation, and multilingual support in English and Greek.

* Fully responsive layout for mobile, tablet, and desktop
* Image carousel with captions
* EN / EL language switcher
* Mobile dropdown navigation menu
* Smooth animations and lightweight performance

Technologies Used

* HTML5
* CSS3
* JavaScript

How to Run

1. Download or clone the project.
2. Open index.html in any web browser.
3. No installation or dependencies required.

Deployment
This site is hosted by Netlify.

Customization

* Edit styles in the CSS file.
* Replace images in the images folder.
* Update text and translations in index.html.
* Modify interactive behavior in the JS script.
